Readme Datei

Aufrufmöglichkeiten über die Konsole:

>>java Main dateipfad s t Algorithmus Mitoutput Outputpfad

dateipfad:
Bitte den Eingabepfad der Datei angeben

s und t:
Bitte als Zahlen angeben

Algorithmus:
Mögliche Eingaben : Beschreibung
0 : kein Algorithmus wird ausgeführt (falls LP File generiert werden soll)
1 : Endmond Karp
2 : Goldberg Tarjan
3 : Erst Endmond Karp und dann Goldberg Tarjan

Mitoutput (optional):
Mögliche Eingaben : Beschreibung
0 : Kein Output
1 : Es gibt eine Outputdatei
2 : Es wird eine LP-file Datei angelegt

Outputpfad (nur wenn Mitoutput=1 oder =2):
Achtung: Bitte den Pfad mit Dateinamen angeben, jedoch NICHT mit einer Endung.
Die Datei wird immer mit ".sol" Endung ausgegeben.
